Готовый простой сайт для App Store: главная страница, Privacy Policy, Support и Terms.

Что заменить перед публикацией:
1. Во всех файлах заменить SUPPORT_EMAIL_HERE на реальный support email.
2. При желании заменить название CES Deck Tanker Practice на финальное название приложения.

Как быстро опубликовать через GitHub Pages:
1. Создать новый public repository на GitHub, например ces-deck-tanker-site.
2. Загрузить в него все файлы из этой папки.
3. Открыть Settings -> Pages.
4. В Source выбрать Deploy from a branch.
5. Branch: main, folder: /root.
6. Нажать Save.
7. Через 1-3 минуты GitHub даст ссылку вида:
   https://USERNAME.github.io/ces-deck-tanker-site/

Что вставлять в App Store Connect:
Privacy Policy URL:
https://USERNAME.github.io/ces-deck-tanker-site/privacy.html

Support URL:
https://USERNAME.github.io/ces-deck-tanker-site/support.html

Marketing URL, если нужен:
https://USERNAME.github.io/ces-deck-tanker-site/
